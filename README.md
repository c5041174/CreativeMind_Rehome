ShareCircle is a Flask + SQLite web app that allows people to give away items they no longer need to others in their community.

## Features

- User authentication
- Post items with images
- Browse available items
- Request items
- Secure password hashing

## Tech Stack

- Backend: Python Flask
- Database: SQLite
- Frontend: HTML, Bootstrap

## Setup

1. Install dependencies:
   pip install flask werkzeug

2. Create database:
   python database.py

3. Run the app:
   python app.py

4. Open in browser:
   http://127.0.0.1:5000/
